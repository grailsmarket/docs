### GET /api/v1/analytics/volume

Sales volume distribution broken down by price range buckets. ETH and WETH sales only.

**Authentication:** None

**Cache:** None

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `period` | string | `7d` | `24h`, `7d`, `30d`, `90d`, `all` |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/analytics/volume"
```
