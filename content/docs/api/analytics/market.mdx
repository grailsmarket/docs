### GET /api/v1/analytics/market

Global market statistics including name counts, active listings/offers, sales volume, and activity metrics.

**Authentication:** None

**Cache:** None

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `period` | string | `7d` | `24h`, `7d`, `30d`, `90d`, `all` |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/analytics/market"
```
