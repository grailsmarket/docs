### GET /api/v1/orders/:id

Get an order by its database ID or Seaport order hash. Both lookup methods are tried.

**Authentication:** None

**Cache:** None

#### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | Listing ID (number) or order hash (hex string) |

#### Sample Request

```sh
curl https://api.grails.app/api/v1/orders/42
# or
curl https://api.grails.app/api/v1/orders/0xabc123...
```

| Status | Code | Description |
|--------|------|-------------|
| 404 | `ORDER_NOT_FOUND` | Order not found |
