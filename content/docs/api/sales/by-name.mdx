### GET /api/v1/sales/name/:name

Get the sales history for a specific ENS name.

**Authentication:** None

**Cache:** 60s

#### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | ENS name (e.g. `example.eth`) |

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | number | `1` | Page number |
| `limit` | number | `20` | Results per page |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/sales/name/example.eth"
```
