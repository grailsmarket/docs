### GET /api/v1/cart

Get the authenticated user's cart items, enriched with full ENS name data (listings, votes, watchers, offers). Optionally filter by cart type.

**Authentication:** Required

**Cache:** None

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `type` | string | — | Filter by cart type name (e.g. `buy`, `register`) |

#### Sample Request

```sh
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "https://api.grails.app/api/v1/cart?type=buy"
```

#### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 400 | `VALIDATION_ERROR` | Invalid query parameters |
| 401 | `UNAUTHORIZED` | Missing or invalid token |
| 500 | `INTERNAL_ERROR` | Failed to fetch cart |
