### GET /api/v1/personas

List all available personas with their descriptions, icons, and default filter configurations for each view.

**Authentication:** None

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/personas"
```

#### Response

```jsonc
// 200 OK
{
  "success": true,
  "data": [
    {
      "slug": "collector",
      "name": "Collector",
      "description": "Focused on accumulating ENS names",
      "icon": "🏆",
      "defaultFilters": {
        "allNames": { "sortBy": "price", "sortOrder": "asc" },
        "listings": { "sortBy": "price", "sortOrder": "asc" },
        "sales": { "sortBy": "date", "sortOrder": "desc" },
        "registrations": { "sortBy": "date", "sortOrder": "desc" },
        "offers": { "sortBy": "price", "sortOrder": "desc" }
      }
    }
  ],
  "meta": {
    "timestamp": "2026-04-08T00:00:00.000Z"
  }
}
```
