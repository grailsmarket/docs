# Personas

User persona classification system. Each persona defines default search filters for different views. Users are automatically classified based on their activity.

**Base path:** `/api/v1/personas`

## Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/` | None | List all personas with default filters |
| GET | `/me` | Required | Get authenticated user's persona and scores |
| POST | `/me/reclassify` | Required | Queue a reclassification for the current user |
