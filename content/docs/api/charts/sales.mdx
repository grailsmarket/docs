### GET /api/v1/charts/sales

Sales count per time bucket with Grails/OpenSea source breakdown.

**Authentication:** None

**Cache:** None

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `period` | string | `7d` | `1d`, `7d`, `30d`, `1y`, `all` |
| `club` | string | — | Filter by club (comma-separated for multiple, e.g. `999,10k`) |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/charts/sales?period=30d&club=999"
```
