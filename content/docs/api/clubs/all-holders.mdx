### GET /api/v1/clubs/holders

Get unique holders aggregated across one or more clubs. If no clubs specified, returns holders across all clubs. Only includes registered and grace-period names.

**Authentication:** None

**Cache:** 15s

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | number | `1` | Page number |
| `limit` | number | `20` | Results per page (1–100) |
| `sortOrder` | string | `desc` | Sort by name count: `asc` or `desc` |
| `clubs[]` | string[] | — | Filter by specific clubs (repeatable) |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/clubs/holders?clubs[]=999&clubs[]=10k&limit=10"
```
