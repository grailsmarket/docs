### GET /api/v1/clubs

List all clubs with metadata, sales statistics, member counts, floor prices, and computed metrics. Supports sorting by 30+ fields, classification filtering, and search.

**Authentication:** None

**Cache:** 5m

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sortBy` | string | `total_sales_volume_wei` | Sort field (see below) |
| `sortOrder` | string | `desc` | `asc` or `desc` |
| `class[]` | string[] | â€” | Filter by classification (repeatable) |
| `search` | string | â€” | Search club name, display name, or description |

#### Sort Fields

**Sales:** `total_sales_count`, `total_sales_volume_wei`, `sales_count_1y`, `sales_count_1mo`, `sales_count_1w`, `sales_volume_wei_1y`, `sales_volume_wei_1mo`, `sales_volume_wei_1w`

**Registrations:** `total_reg_count`, `total_reg_volume_wei`, `reg_count_1y`, `reg_count_1mo`, `reg_count_1w`, `reg_volume_wei_1y`, `reg_volume_wei_1mo`, `reg_volume_wei_1w`

**Members:** `member_count`, `registered_count`, `grace_count`, `premium_count`, `available_count`, `listings_count`, `registered_percent`, `grace_percent`, `premium_percent`, `available_percent`, `listings_percent`

**Holders:** `holders_count`, `holders_ratio`

**Other:** `floor_price_wei`, `name`

#### Classifications

`ethmojis`, `digits`, `palindromes`, `prepunk`, `geo`, `letters`, `fantasy`, `crypto`, `ai`

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/clubs?sortBy=floor_price_wei&sortOrder=desc&class[]=digits"
```
