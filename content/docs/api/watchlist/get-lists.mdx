### GET /api/v1/watchlist/lists

Get all of the authenticated user's watchlists with item counts. The default list is returned first.

**Authentication:** Required

**Cache:** None

#### Sample Request

```sh
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.grails.app/api/v1/watchlist/lists
```
