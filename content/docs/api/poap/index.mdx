# POAP

POAP (Proof of Attendance Protocol) link distribution. Each user can claim one POAP link.

**Base path:** `/api/v1/poap`

## Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| POST | `/claim` | Required | Claim a POAP link (one per user) |
| GET | `/status` | Required | Check if current user has claimed |
| GET | `/stats` | None | Get total/claimed/remaining counts |
