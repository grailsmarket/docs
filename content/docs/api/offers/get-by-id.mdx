### GET /api/v1/offers/:id

Get a single offer by its ID.

**Authentication:** None

**Cache:** None

#### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | number | Yes | Offer ID |

#### Sample Request

```sh
curl https://api.grails.market/api/v1/offers/99
```
