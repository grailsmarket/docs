# Brokered Listings

Listings created by a broker on behalf of a seller, with a configurable broker fee. The broker fee is encoded in the Seaport order's consideration items and validated server-side.

**Base path:** `/api/v1/brokered-listings`

## Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/config` | None | Get broker fee configuration |
| POST | `/` | Required | Create a brokered listing |
| GET | `/broker/:address` | None | Get listings by broker address |
| GET | `/stats` | None | Get aggregate broker statistics |
