### WS /ws/orders

Order-specific WebSocket channel for real-time order status updates.

#### Connection Example

```sh
wscat -c "wss://api.grails.app/ws/orders"
```

On connect, the server sends:

```jsonc
{ "type": "connected", "channel": "orders", "clientId": "req-1", "timestamp": "2026-04-08T00:00:00.000Z" }
```

#### Client Messages

| Message Type | Fields | Description |
|--------------|--------|-------------|
| `subscribe` | `orderId`: string | Subscribe to updates for a specific order |

#### Server Messages

| Message Type | Description |
|--------------|-------------|
| `connected` | Connection established (includes `channel`, `clientId`) |
| `subscribed` | Subscription confirmed (includes `orderId`) |
| `error` | Error message |
