### GET /api/v1/trending/sales

Trending names ranked by sales activity in the period.

**Authentication:** Optional

#### Trending Metrics

| Field | Type | Description |
|-------|------|-------------|
| `period_sales` | number | Number of sales in the period |
| `period_volume` | string | Total sales volume in wei |
| `avg_price` | string | Average sale price in wei |
| `max_price` | string | Highest sale price in wei |
| `min_price` | string | Lowest sale price in wei |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/trending/sales?period=7d&limit=10"
```
