### GET /api/v1/trending/votes

Trending names ranked by total voting activity (upvotes + downvotes) in the period.

**Authentication:** Optional

**Cache:** None

#### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `period` | string | `24h` | Time period: `24h` or `7d` |
| `limit` | number | `20` | Results to return (1-100) |

#### Trending Metrics

| Field | Type | Description |
|-------|------|-------------|
| `period_upvotes` | number | Upvotes in the period |
| `period_downvotes` | number | Downvotes in the period |
| `period_votes` | number | Total votes in the period |
| `net_score_total` | number | All-time net score |

#### Sample Request

```sh
curl "https://api.grails.app/api/v1/trending/votes?period=24h&limit=10"
```
